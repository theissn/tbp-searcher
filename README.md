# TPB Searcher

Simple cli tool to search TPB and open in your preferred torrent client.

## Demo

[Video](./docs/demo.mp4)

## Usage

```
tpb-searcher SEARCH_QUERY
```

## Build

```
git clone github.com:theissn/tpb-searcher
cd tpb-searcher
go build
```

### Install

```
go install github.com/theissn/tpb-searcher@latest
```

### Dev

```
git clone github.com:theissn/tpb-searcher
cd tpb-searcher
go run . SEARCH_QUERY
```
