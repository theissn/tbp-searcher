# TPB Searcher

Simple cli tool to search TBP and open in your preferred torrent client.

## Demo

[Video](./docs/demo.mp4)

## Usage

```
tbp-searcher SEARCH_QUERY
```

## Build / Install

```
git clone github.com:theissn/tbp-searcher
cd tbp-searcher
```

### Install

```
go install
```

### Dev

```
go run . SEARCH_QUERY
```
